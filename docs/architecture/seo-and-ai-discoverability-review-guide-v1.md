# Mentor IB SEO And AI Discoverability Review Guide v1

**Date:** 2026-04-08
**Status:** Review companion for `seo-and-ai-discoverability-v1.md`

## 1. Why This Guide Exists

This guide helps review the SEO and AI-discoverability direction quickly.

## 2. Review Goal

The goal is to decide whether Mentor IB has the right discoverability architecture before implementation starts.

This is not yet a keyword-plan review.

## 3. Best Sections To Review First

Start here in [seo-and-ai-discoverability-v1.md](seo-and-ai-discoverability-v1.md):

- `Decisions To Lock Now`

## 4. Core Review Questions

1. Does the doc correctly treat AI discoverability as an extension of strong SEO, not a separate hack layer?
2. Is the recommended indexable page set realistic for phase 1 and phase 1.5?
3. Is the non-indexable page policy strict enough for personalized and operational surfaces?
4. Does the rendering guidance fit the approved Next.js architecture?
5. Is the structured data plan conservative and trustworthy enough?
6. Does the proposed public page inventory align with Mentor IB's matching-first product model?
